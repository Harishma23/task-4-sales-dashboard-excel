# Task 4: Sales Dashboard Design

## 📊 Overview
This project is part of the Data Analyst Internship Task 4.  
The objective is to create an interactive dashboard using sales data.

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Charts (Line, Bar, Column)
- Slicers

## 📁 Dataset
- Sales dataset (CSV file) with Date, Region, Category, Sales, and Profit.

## 📈 Dashboard Features
- Sales trend over time
- Sales comparison by region
- Profit analysis by category
- Interactive filtering using slicers

## 🎯 Outcome
The dashboard helps analyze sales performance, identify trends, and support business decisions.

## 📌 Files Included
- sales_data.csv
- Excel dashboard file (.xlsx)
- Dashboard screenshot
- Presentation (PPT)
